# Titanic - Análise Exploratória de Dados

Este projeto realiza uma análise exploratória do dataset **Titanic**, utilizando as bibliotecas **Pandas** e **NumPy** em Python.

## Objetivos
- Investigar os dados de sobrevivência dos passageiros.
- Responder à pergunta: **"Mulheres e crianças primeiro?"**
- Explorar a curiosidade cultural: **Jack e Rose existiram?**

## Tecnologias Utilizadas
- Python 
- Pandas
- NumPy
- Jupyter Notebook

## Perguntas de Pesquisa
1. Qual foi o impacto do gênero e da idade na taxa de sobrevivência?  
2. A regra "mulheres e crianças primeiro" se confirma nos dados?  
3. Existe alguma evidência sobre os personagens Jack e Rose?
